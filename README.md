# Numerical methods of solving nonlinear equations $f(x)=0$
Implementation of main numerical methods for solving non-linear equations f(x)=0 with examples.

1. Bisection method
2. Secant method
3. Newton's method
4. Fixed point method

## Requirements
- Python 3.x
- Jupyter Notebook
- Matplotlib
